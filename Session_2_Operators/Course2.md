# 🧩 C# `try-catch` 錯誤處理教學

## 🔹 什麼是 `try-catch`

在 C# 中，`try-catch` 用來**處理執行期間的例外狀況 (Exception)**。
這樣即使程式發生錯誤，也不會中斷執行，能讓你優雅地應對錯誤。

---

## 🔸 基本語法

```csharp
try
{
    // 嘗試執行可能會發生錯誤的程式碼
}
catch (Exception ex)
{
    // 當錯誤發生時執行這裡
    Console.WriteLine("錯誤訊息：" + ex.Message);
}
```
